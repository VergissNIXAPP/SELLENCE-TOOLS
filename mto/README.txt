SELLENCE – MTO Rechner (Web)
---------------------------

Datei: mto.html

- Öffne die Datei lokal im Browser ODER hoste sie wie deine anderen SELLENCE-TOOLS Seiten (Vercel / SharePoint / etc.).
- Die Gewichtungen kommen direkt aus deiner Excel (Sheet "Gewichtung").
- Die Preis-Konstanten kommen aus den Formeln im Sheet "Gebinde & Stückzahlen = Monat".

Einbindung in SELLENCE-TOOLS:
- Einfach als eigenes Tool verlinken (z.B. Button "MTO Rechner" -> mto.html)

Hinweis:
In der Excel-Formel für OTP stand "… + E30 + 2.87". Das wirkt wie ein Tippfehler.
Hier ist es sinnvoll umgesetzt als: (Cigarillos-Anzahl × 2,87 €).
